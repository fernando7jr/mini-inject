# Reporting Security Issues

If by any chance you have discovered a security vulnerability in our project, we take it very seriously and appreciate your help in keeping our software safe. Please open an issue in our GitHub repository and provide as much detail as possible about the vulnerability, including steps to reproduce it.

This package is meant for community use and there is zero obfuscation or blackbox logic in the code. For any questions or concerns, please reach out to us directly through our GitHub repository. We do not provide rewards or bounties for security issues, as this is an open source passion project.
